# lit-ssg-transform

Render Lit web components in SSG outputs using `@lit-labs/ssr`.

> **⚠ WARNING: Experimental - no security or stability guarantees**

```sh
npm install && npm run build
node dist/cli.js
```

Scans the output of a static site for custom elements and renders them to static HTML.

```sh
lit-ssg --dir public
```
